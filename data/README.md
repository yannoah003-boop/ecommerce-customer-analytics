## Dataset

This project uses the **Online Retail Dataset** available on Kaggle.

Dataset link:
https://www.kaggle.com/datasets/ulrikthygepedersen/online-retail-dataset/data

### Dataset Description

The dataset contains transactional data from a UK-based online retail store between **December 2010 and December 2011**.  
The company mainly sells unique all-occasion gifts and many customers are wholesalers.

Each row represents a single transaction line item.

### Key Fields

The dataset includes the following main columns:

- **InvoiceNo** – Invoice number for each transaction
- **StockCode** – Unique product/item code
- **Description** – Product description
- **Quantity** – Number of items purchased
- **InvoiceDate** – Date and time of the transaction
- **UnitPrice** – Price per item
- **CustomerID** – Unique identifier of the customer
- **Country** – Country of the customer

### Note

The raw dataset is **not included in this repository due to file size limitations**.

You can download the dataset directly from Kaggle using the link above.
